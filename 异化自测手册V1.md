# 异化自测手册 v1

你不是懒，你是被异化了。

本手册帮助你识别：你每天的学习、劳动、休闲，到底是“被迫的消耗”还是“主动的创造”。请根据你的身份，选择对应的部分自测。

---

### 一、六个词（所有人都适用）

| 缩写 | 全称 | 一句话解释 |
|---|----------|---------------------------------------------------
| AS | 异化学习 | 不想学，但被逼着学 |
| BS | 归化学习 | 自己想学，主动去学 |
| AA | 异化劳动 | 不想干，但不得不干 |
| BA | 归化劳动 | 想干，干完有成就感 |
| UP | 麻醉式休娱 | 刷短视频、打游戏、喝酒，当时爽，事后空虚
| DP | 平静休娱 | 发呆、散步、喝茶，不逃避，只是空一会儿 

---

### 二、中学生版

自测题（回想最近一周）

1.上课时
   - [ ] AS（人在心不在，脑子里想别的）  
   - [ ] BS（这节课我感兴趣，主动听）  

2.写作业时
   - [ ] AA（不想写，但必须交）  
   - [ ] BA（写完有成就感，或者能学到自己想学的东西）

3. 放学后，你通常
   - [ ] UP（刷手机、打游戏，一刷几小时，关掉更空虚）  
   - [ ] DP（发呆、散步、看星星，什么都不想）  

4. 周末或假期，你主动做的事
   - [ ] 几乎没有，只想躺着（UP/AA）  
   - [ ] 有一件你愿意花时间、不觉得累的事（BA/BS）  

结果：如果AS+AA+UP ≥ 3个 → 异化程度较高。不是你懒，是系统在推你。  
建议：找到一件你的“归化劳动”（BA）——哪怕只是每天画10分钟画、写几行日记。那是你对抗虚无的工具。

---
### 三、打工人版

自测题（回想最近一周）

1. 工作时
   - [ ] AA（不想干，但为了工资不得不干）  
   - [ ] BA（有些任务做完有成就感，或者能学到东西）  

2. 加班时
   - [ ] AA（被迫加班，效率低下）  
   - [ ] BA（主动留下来做自己负责的项目，有进展）  

3. 下班后，你通常
   - [ ] UP（刷短视频、打游戏、喝酒，停不下来）  
   - [ ] DP（散步、做饭、陪家人，不逃避）  

4. 周末或假期，你主动做的事
   - [ ] 几乎没有，只想补觉（UP/AA）  
   - [ ] 有一件你愿意花时间、不觉得累的事（BA/BS）  

结果：如果AA+UP ≥ 2个 → 异化程度较高。你感到累、空虚、没意义，不是你的错，是系统在推你。  
建议：每天留30分钟给“归化劳动”（BA）——哪怕只是做一道菜、修一张图、写一段文字。那是你夺回控制权的第一步。

---

### 四、家长版

自测题（回想最近一周）

1. 辅导孩子作业时
   - [ ] AA（不想管，但不得不管，容易发火）  
   - [ ] BA（愿意陪孩子一起解决问题，有耐心）  

2. 做家务时
   - [ ] AA（重复枯燥，做不完，心累）  
   - [ ] BA（整理完有清爽感，或愿意尝试新菜谱）  

3. 孩子睡着后，你通常
   - [ ] UP（刷手机、追剧，报复性熬夜）  
   - [ ] DP（泡杯茶，静一会儿，或者和伴侣聊几句）  

4. 你自己主动想做的事（非家庭责任）
   - [ ] 几乎没有（UP/AA）  
   - [ ] 有几件（养花、运动、看书、学东西……）

结果：如果AA+UP ≥ 3个 → 异化程度较高。你的疲惫不是“不够坚强”，是结构性的。  
建议：哪怕每天15分钟，做一件只属于你、不为了别人的事。那不是自私，是你不被异化的证据。

---

### 五、下一步（所有人）

找到一件你的“归化劳动”（BA）
——那件没人逼你、你愿意主动做、做完不空虚的事。  

每天给它30分钟（中学生/打工人）或15分钟（家长）。  
不需要多，只需要持续。这不是“成功学”，这是你对抗虚无的钢锯。
