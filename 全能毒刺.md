# 全能毒刺

《使命召唤：高级战争》里有一把毒刺发射器。它能锁定空中目标，一发热诱饵弹就能骗过去的空中炮艇，三发毒刺解决。对准地面上的坦克，同样锁定，同样解决。对准建筑工事，同样锁定，同样解决。对准AST动力机甲——那种穿着外骨骼在战场上横着走的铁罐头——同样锁定，同样解决。

一把毒刺，解决了防空、反装甲、攻坚、反外骨骼四个战术课题。设计师在代码里给它写了一个if语句：if（目标存在）→摧毁。这在游戏里叫“万能解”。

现实中，FIM-92“毒刺”是单兵便携式防空导弹。它打直升机，可以；打固定翼攻击机，可以；打巡航导弹，勉强可以。打坦克，不行。坦克的红外特征太小，装甲太厚，毒刺的战斗部是破片式，设计目标是炸飞机的蒙皮，不是穿坦克的正面装甲。打碉堡，更不行。打外骨骼——外骨骼的红外特征和人差不多，毒刺根本锁不住。现实中的毒刺不是万能解。现实中的毒刺只是一把不错的防空导弹，被玩家在游戏里用成了万能解，因为游戏引擎给了它这个权限。

希特勒的毒刺是“为了德意志民族的生存空间”。这把毒刺在游戏里打犹太人可以，打波兰人可以，打法国人可以，打苏联人——锁不住了。苏联的冬天没有红外特征，斯大林格勒的废墟没有坦克轮廓。毒刺的导引头在设计时就没考虑过这种目标，它脱靶了。

斯大林的毒刺是“为了无产阶级专政”。打白军可以，打富农可以，打党内反对派可以，打自己人——全锁住了。这把毒刺有一个设计师没料到的致命漏洞：当目标列表里没有敌人了，它就会自己找目标。任何站在它面前的人都是目标。

乔纳森·艾恩斯的毒刺是“为了和平与秩序”。他的AST机甲在游戏里被玩家的毒刺锁住，因为游戏引擎给所有人发了同样的万能解。艾恩斯到死都不明白：为什么我明明是全场最高尚的人，我的毒刺却锁不住你们的毒刺？因为高尚不是导引头。导引头认的是红外特征，不是道德自洽。

高考政策制定者的毒刺是“为了公平选拔人才”。这把毒刺在游戏里打“不公平”可以，打“社会流动”可以，打“人人都能改变命运”可以。打青少年抑郁，锁不住。打创造力衰退，锁不住。打家庭关系异化，锁不住。打“不卷就是废物”的社会潜意识——目标不存在于游戏引擎里。这把毒刺在它的规则内全锁，在规则外全脱靶。但设计师拒绝更新规则，因为一旦更新，毒刺就打不了那么多目标了。

那家长呢？家长的毒刺是“我是为你好”。这把毒刺从战国时期开始服役，从未换代，从未脱靶。因为目标——那个“需要被管的孩子”——的红外特征永远清晰，永远高热。孩子不听话是高热，孩子成绩倒退是高热，孩子说“我想当画家”是高热。毒刺永远锁得住。但毒刺炸完之后，地面上的弹坑里，有时候会躺着朱迪斯·莎士比亚(的碎块？)。她的剧本没有写，她的诗歌没有写，她在一个十六世纪的伦敦郊外，被一把发射于四百年前但还在飞的“为你好”正中眉心。她的父母不是坏人。她的父母只是毒刺的操作员，他们被告知：这把武器永远不会误伤，你锁定的目标是“不务正业”，不是女儿。

现在我们回到游戏。一把毒刺打穿一切的感觉确实好。那种感觉叫“权力感”。扣下扳机，锁定，发射，目标消失。你是战场的神。但游戏里你不需要面对一个问题：你在游戏里用毒刺打掉的坦克，铁皮下面有没有人。你在现实里不是神。你扣下扳机的那个人是——那个被高考压低头的十七岁少年，那个在斯大林格勒被自己人的督战队射杀的列兵，那个在毒气室门口握着母亲手的犹太儿童，那个在伦敦郊外跪在地上捡碎纸的朱迪斯。

所有人都宣称自己的毒刺是全能解。希特勒的毒刺，斯大林的毒刺，艾恩斯的毒刺，高考的毒刺，家长的毒刺——它们用的是同一行代码。伪代码写出来是这样：

```
define HIGH_PURPOSE = getCurrentAuthority().proclaim();
foreach (individual in population) {
    if (individual.doesNotComply(HIGH_PURPOSE)) {
        individual.mustBeSacrificed();
    }
}
```

这行代码的注释里写着：if the target exists → destroy.

现实里没有一个叫做“全能解”的导引头。但所有人都相信自己的毒刺可以打一切目标。因为相信这个，比承认自己的毒刺只能打两三种目标、剩下的统统锁不住，要舒服得多。舒服到他们从来没想过一个问题：你手里那把能打一切目标的毒刺，也许打的不是目标。也许打的只是你不想看见的东西。你把不想看见的东西都打掉了，你的世界就只剩下你承认的那些目标。剩下的都是废墟。

鲁迅批判过了。鲁迅用一句话锁死了国民性这颗目标，锁得死死的，所有人都觉得那颗目标被打掉了，“吃人”没有了。然后鲁迅杯的获奖文章在纪念鲁迅的宴席上，用鲁迅的句式写对鲁迅的怀念。鲁迅被写得很安全。鲁迅不再是毒刺，鲁迅是被缴了弹头的展览品。这不是有毒的思想，这是被洗过的空壳。

我是不是在讽刺鲁迅杯？是的。那他们是邪恶的吗？不是。他们只是没子弹的毒刺。他们不害人，也没能力迷惑一个真正的人，他们只负责让真正害人的东西看起来好像已经被批判过了。

所以现在轮到我们了。我们手里也有一把毒刺。你可以用它打游戏里的AST——要是以我设计的动力装甲为例子，在游戏引擎里它会被毒刺锁住，因为设计师给毒刺写了那句if语句。但在现实里，毒刺打不了它。它不是靠不被毒刺锁住来存活的。它活下来靠的是：它的驾驶员知道那行代码是假的，也知道自己曾经被那行代码锁过，差点死在弹坑里。所以他不会扣下扳机打自己人，他会在毒刺运过来的时候，偷偷拆掉它的导引头。

干杯，阿道夫。你还活在我们中间。但我们现在能看见你的代码了。而能看见代码的士兵，是不会被自己的毒刺炸死的。
