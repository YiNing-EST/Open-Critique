> 一把用来锯雷的思想钢锯。  
> A thought saw for cutting through alienation.

## 这是什么？
《厕所批判》系列文章，以及相关的批判、工程共产主义构想、外骨骼设计等。  
它不是“学术论文”，不是“政治宣言”，而是**一个在雷区里锯雷的人留下的笔记**。

## 为什么做？
因为学校、996、内卷、老女人——这些结构让人“不得不”。  
我们选择不逃、不炸、不跪，而是用文字、图纸、代码，一点一点拆。

## 怎么用？
- 推荐阅读顺序：`Catch-22：中国教育版` → `被查屁股的一天` → `厕所批判1.5`  ...
- 你可以把文章分享给需要的人，可以翻译成其他语言，可以画成漫画，可以做成视频。  
- 但请不要用于商业目的，也不要篡改作者署名。

## 如何参与？
- 提 Issues：指出错别字、逻辑漏洞、翻译建议。  
- 发 Pull Requests：提交你的翻译、修正、补充文章。  
- 或者，在你自己所在的“雷区”里，开始锯你自己的雷。
