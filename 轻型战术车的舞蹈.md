# 轻型战术车的舞蹈

在游戏《三角洲行动》里，有一种车。它没有车门，车窗的玻璃薄得像是用透明胶带糊上去的。车身的装甲也薄，薄到什么程度呢？AT-4一发过来，它就变成一团火球。但如果你没被炸，它跑起来是轻快的，灵巧的，像一个在开阔地上撒欢的金属动物。

我学会了一件事：开这辆车，绝对不能停。

不能停，不是因为停不下来，是因为一旦停下，对方就有足够的时间锁定你，瞄准你，然后送你一发标枪或火箭弹。你只能一直跑，一直动，一直让那两吨重的铁皮在你和子弹之间做无规则的布朗运动。

这让我想起一个作家的短篇里那句话——
“舞舞舞，只要还在跳舞，就不会掉下去。”

我忘了那个作家叫什么名字了，也可能记混了。也许是村上春树，也许不是。但这都不重要。重要的是那句话是对的。

你在那辆车上，就是在跳舞。你的舞伴是那些AT-4和坦克炮弹，你的舞步是油门和方向盘的配合。你一旦停下来，舞会就结束了。子弹会赶在你重新起步之前，精准地把你从舞池里踢出去。

这是一种很奇怪的东西：跑，不是为了到达，而是为了不被击中。你的存在，在那些交战的瞬间，纯粹是为了保持移动。你没有一个需要抵达的目的地，你只有一个需要逃离的现在时。

那个不会让你停下来的舞步，其实是一种策略。移动不是为了到达任何地方，而是为了证明你还没被击中。

让我再想一想，那个到底是不是村上春树写的。我记得有一本小说，主人公在梦里跳舞，在现实里弄丢什么东西，然后绕了很大一圈，最后在一个很安静的、只有海风和星光的地方找到了那个东西。不是《舞！舞！舞！》吗？还是别的什么？那个作家叫什么来着？

算了，这不重要。重要的是：开那辆车，不能停。它教你的不是如何战斗，而是如何存在。
